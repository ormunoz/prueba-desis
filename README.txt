Instalación de aplicaciones necesarias

Se requieren las siguientes aplicaciones para el funcionamiento del proyecto

-XAMPP: para que actúe de servidor de la aplicación web
-(opcional) Visual Studio Code o cualquier editor de texto de preferencia.

1) A continuación se deberá de colocar la carpeta del proyecto en la dirección "C:\xampp\htdocs".
2) en la carpeta SQL estará el archivo de la base de datos, esta se debe importar en el la ruta http://localhost/phpmyadmin/
3) presionamos click en nueva y creamos una nueva base de datos.
4) luego en la base de datos creada, seleccionaremos la pestaña importar, seleccionar archivo y ponemos el archivo db_votacion.sql, y apretamos importar.
5) para finalizar ingresaremos a la siguiente ruta http://localhost/desis/src/view/indexView.php y podremos interactuar con la aplicación
